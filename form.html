input属性 其他相关属性 解释
text 文本框
password 密码框
radio 单选按钮
  • name属性 : 表示单选按钮所在的组名
  • value属性 : 表示单选按钮的取值
  • checked属性 : 表示默认选中，可缩写
  • 对于一组单选按钮，只能选中其中一项
checkbox 复选框
  • name属性 : 表示单选按钮所在的组名
  • value属性 : 表示单选按钮的取值
  • checked属性 : 表示默认选中，可缩写
  • 复选框可以选择多项
button 普通按钮 一般配合js使用
submit 提交 用于给服务器提交数据的
reset 重置 重置按钮只能清空它所在form标签内的表单的内容
file 文件上传 当单机选择文件按钮后,会发现不能上传文件,这个需要后端知识
